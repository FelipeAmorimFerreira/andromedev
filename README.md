## Desenvolvimento

```sh
yarn && yarn develop
```

Acesse http://localhost:8000/

Caso haja algum problema de build, antes de reportar, rode esse comando dentro do repositório e tente ver se o problema é resolvido:

```sh
yarn clean
```
